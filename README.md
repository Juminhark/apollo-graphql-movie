# apollo-graphql-movie

Make GraphQL Server - Movie API with GraphQL

## Stack

- server : [graghql-yoga](https://github.com/prisma-labs/graphql-yoga)
- data layer : apollo-graphql
- client : [create-react-app](https://create-react-app.dev/)

```sh
code .
```

## Client
